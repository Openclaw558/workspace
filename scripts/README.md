## 🔧 Usage

### DOCX Extraction

Extract content dari Microsoft Word (.docx) files.

```bash
python3 extract-docx.py <file.docx>
```

**Example:**
```bash
python3 scripts/extract-docx.py docs/knowledgeChronicle.docx
```
